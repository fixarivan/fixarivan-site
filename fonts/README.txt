Папка для шрифтов DomPDF
========================

Для корректной работы кириллицы в PDF необходимо поместить сюда файл:
- DejaVuSans.ttf (или DejaVuSansCondensed.ttf)

Скачать можно с: https://dejavu-fonts.github.io/

Альтернативно можно использовать:
- Roboto-Regular.ttf
- OpenSans-Regular.ttf

Все шрифты должны поддерживать кириллицу (UTF-8).
