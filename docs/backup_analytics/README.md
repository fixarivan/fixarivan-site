# Аналитика в архивах бэкапа

Эта папка попадает в zip, который создаёт `tools/backup.php` (вместе с `storage/`).

**Назначение для ИИ и людей**

- **`ANALYTICS_REPORT.md`** — сжатый аналитический отчёт о доработках бизнес-логики (счета, валидация, языки, PDF, валюта, viewer).
- При разборе бэкапа без репозитория сначала читайте этот файл, затем пути из раздела «Ключевые файлы».

Обновляйте `ANALYTICS_REPORT.md` при существенных изменениях пайплайна счетов/документов.
