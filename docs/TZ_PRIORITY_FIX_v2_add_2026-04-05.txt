Усилить существующее ТЗ, устранив недосказанности и зафиксировав жёсткие правила

================================================================================
БЛОК A — ЗАПРЕТ FALLBACK ПО ИМЕНИ (КРИТИЧНО)
--------------------------------------------------------------------------------
Если отсутствует inventory_item_id:
- он должен быть создан ДО выполнения любой операции
- операция не выполняется через name

Запрещено:
- использовать имя позиции как источник истины

Разрешено:
- использовать имя только для поиска в UI

================================================================================
БЛОК B — РАЗДЕЛЕНИЕ ИДЕНТИФИКАТОРОВ
--------------------------------------------------------------------------------
Чётко зафиксировать:

inventory_item_id:
- идентификатор складской позиции

order_warehouse_line_id:
- идентификатор строки заказа

Правило:
- операции выполняются через order_warehouse_line_id
- inventory_item_id используется для связи со складом

================================================================================
БЛОК C — ОБЯЗАТЕЛЬНЫЙ ORDER_ID
--------------------------------------------------------------------------------
Все действия должны выполняться с order_id:

Если order_id отсутствует:
- операция не выполняется
- пользователь должен выбрать заказ

Запрещено:
- угадывать заказ
- использовать "последний заказ клиента"

================================================================================
БЛОК D — ЗАПРЕТ СОЗДАНИЯ ДУБЛЕЙ ЗАКАЗА
--------------------------------------------------------------------------------
Если действие происходит внутри заказа:
- использовать текущий order_id
- новый заказ НЕ создаётся

================================================================================
БЛОК E — ИСТОЧНИКИ ДАННЫХ ФИНАНСОВ
--------------------------------------------------------------------------------
Доход:
- sale_price * qty

Расход:
- purchase_price * qty

Работа:
- estimated_labor_cost (число)

Запрещено:
- использовать TEXT для расчётов
- использовать только агрегаты без строк
